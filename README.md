# team-sync-node
IOjs version of the Team Sync API for comparison

## Dependencies
* Rethink DB installed locally
* IOjs 1.3 or higher

## RethinkDB Setup
* Install RethinkDB locally
* Create db 'teamsync'
* Create Tables 'users', 'questions', 'reports', 'teams'

## Initial Run
* `git clone git@github.com:chriscantu/team-sync-node.git`
* `cd team-sync-node`
* `npm install`
* `node app/app.js`
